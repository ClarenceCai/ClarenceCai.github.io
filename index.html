Projects

Hand Pose Estimation
Zikui Cai

Developed an algorithm for human hand pose estimation based on the method of Openpose library. Accelerated the postprocess by putting gaussian filtering operation in Keras model.

2D Skeleton-Based Dance Genre Recognition
Zikui Cai, Ruiwen Zhao, Xiaodi Fan

Made a dataset by crawling street dance videos from Youtube, cleaned and labelled them. Classified videos using CNN & RNN based on extracted human body skeleton information. Generalized the prior arts to solve real life problems in 2D space.

Human Pose Estimation
Zikui Cai, Ruiwen Zhao

Built an VGGNet based 2-branch multistage CNN model using Keras. Trained and validated the model on COCO dataset and got a good result. Compared this bottom-up method with other top-down methods. Gained a better understanding about feature extraction using CNN.

Autonomous Vehicles
Zikui Cai, Vu Tran

Designed a system for vehicle remote controlling and image capturing using Raspberry Pi 3B. Processed pictures captured by cameras that are mounted on vehicles with OpenCV. Utilized CNN model to train the vehicles to navigate on certain tracks.
